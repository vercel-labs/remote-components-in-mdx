# Remote Components in MDX demo

<br />

Below is a demo of shadcn components being rendered in an MDX file.

<br />

Inside of the MDX component the user writes:

<br />
```
<RemoteComponent src="http://localhost:4000/remote-components/button-demo" />
```
<br />
And the following is rendered:

<RemoteComponent src="http://localhost:4000/remote-components/button-demo" />
